# Vagrant QuickStart

This is a Vagrant Environment QuickStart Project. Which help you to quickstart your environment with Vagrant.

## Example

### Go to the space you like

```shell
cd centos7
# OR
cd centos8
```

### Play with Vagrant

```shel
# (启动虚机)
vagrant up

# (连接虚机 -- SSH)
vagrant ssh

# (停止虚机)
vagrant halt

# (销毁虚机)
vagrant destroy -f
```
