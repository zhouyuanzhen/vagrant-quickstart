# README

This is a quick start vagrant box environment with Jupyter Notebook service running in background.

## QuickStart

```shell
# (Spawn up this vagrant python Jupter Notebook box)
$ vagrant up

$ vagrant ssh
(.venv) [vagrant@centos7python ~]$

...

```
